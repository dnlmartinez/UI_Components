# CRAZY DESIGN HELPER

![](igor_8h_after.gif)

## Installation

Use the package manager 

```bash
# New APP:
npx create-react-app IGORAPP
cd IGORAPP
```

## Add Storybook

```bash
#Storybook:
npx -p @storybook/cli sb init
```

## Run the server

```
npm run storybook
```


## More Commands

```
# run terminal test:
yarn test

# init component finder in port 9009:
yarn run storybook

# Run frontend in port 3000:
yarn start
```
